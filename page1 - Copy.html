Python

Python je programski jezik opšte namjene, interpretiran i visoke razine kojeg je stvorio
Guido van Rossum 1990. godine,ime dobiva po televizijskoj seriji Monty Python's Flying Circus.
Po automatskoj memorijskoj alokaciji, Python je sličan programskim jezicima kao što su Perl, Ruby itd.
Python dopušta programerima korištenje nekoliko stilova programiranja. Python je trenutno popularan
